About Me

Interactive infographics

My name is Aneta Stojanowska and I create interactive infographics. That means no static pdf’s, but highly visual, interactive content. As an infographic web designer I translate reports and figures into engaging content. Branded infographics, visual statistics, interactive timelines, you name it.
